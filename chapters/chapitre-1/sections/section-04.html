Racines à partir de la forme développée

Déterminer les solutions de l’équation $f(x) = 0$ pour un trinôme de la forme $ax^2 + bx + c$ à l’aide du discriminant.

Méthodes et exemples

Cette section présente la méthode générale pour trouver les racines d’un trinôme, ainsi que quelques astuces qui permettent de simplifier les calculs.

  1. Cas régulier

    Trouver les racines d’un trinôme de la forme $ax^2 + bx + c$ revient à résoudre l'équation $ax^2 + bx + c = 0$.

    On rappelle la formule pour résoudre une équation du second degré :

    $\Delta = b^2 - 4ac$

    $x_1 = \frac{-b + \sqrt{\Delta}}{2a}, \quad x_2 = \frac{-b - \sqrt{\Delta}}{2a}$

    Si, par exemple, on doit trouver les racines de la fonction:

    \begin{aligned} f(x)=-x^2+4x+5 \end{aligned}

    On résout:

    \begin{aligned} \Delta &= b^2 - 4ac = 4^2 - 4(-1)(5) = 16 + 20 = 36, \\n x_1 &= \frac{-b + \sqrt{\Delta}}{2a} = \frac{-4 + \sqrt{36}}{2(-1)} = \frac{-4 + 6}{-2} = -1, \\n x_2 &= \frac{-b - \sqrt{\Delta}}{2a} = \frac{-4 - \sqrt{36}}{2(-1)} = \frac{-4 - 6}{-2} = 5.\n \end{aligned}

    Les racines sont donc $-1$ et $5$.

  2. Cas particuliers

    Certains cas particuliers permettent de conclure plus rapidement.

    1. $\Delta = 0$

      Si, au cours de la résolution, on obtient $\Delta = 0$, alors l'équation admet une racine double donnée par $x = \frac{-b}{2a}$.

      Par exemple, pour $f(x) = x^2 - 2x + 1$ :

      \begin{aligned} \Delta &= (-2)^2 - 4(1)(1) = 0, \\n x &= \frac{-(-2)}{2(1)} = 1.\n \end{aligned}

      La racine double pour cette fonction est donc $1$.

    2. $a+b+c = 0$

      Si $a + b + c = 0$, alors les racines sont $x_1 = 1$ et $x_2 = \frac{c}{a}$.

      Par exemple, pour $f(x) = 2x^2 - 3x + 1$, on obtient $a + b + c = 2 - 3 + 1 = 0$ ; on a donc $x_1 = 1$ et $x_2 = \frac{1}{2}$.

    3. $a-b+c=0, \quad b=a+c$

      Si $a - b + c = 0$ (ce qui équivaut à $b = a + c$), alors les solutions sont $x_1 = -1$ et $x_2 = -\frac{c}{a}$.

      Par exemple, pour $f(x) = 2x^2 + 3x + 1$, on observe que $a - b + c = 2 - 3 + 1 = 0$ ; les racines valent donc $x_1 = -1$ et $x_2 = -\frac{1}{2}$.
