Chapitre 1 · Section 16

Factorisation d’un polynôme de degré 3

Factoriser un polynôme cubique $P(x)$ lorsque l’on connaît une racine évidente, en appliquant la division euclidienne puis en étudiant le trinôme quotient.

Accueil › Chapitre 1 › Section 16

Objectifs de la section

  • Utiliser la racine connue pour factoriser $P(x)$.
  • Effectuer une division euclidienne par $(x - r)$ ou utiliser Horner.
  • Résoudre le trinôme quotient pour obtenir la factorisation complète.
  • Vérifier la factorisation en redéveloppant.

Procédure détaillée

  1. Identifier la racine

    Si $P(r) = 0$, alors $(x - r)$ est un facteur de $P(x)$.

  2. Diviser le polynôme

    Effectuer la division euclidienne de $P(x)$ par $(x - r)$ pour obtenir $P(x) = (x - r)Q(x)$ où $Q$ est un trinôme.

    On peut utiliser la méthode de Horner pour accélérer le calcul.

  3. Étudier le trinôme quotient

    Résoudre $Q(x) = 0$ en appliquant les méthodes des sections précédentes.

  4. Écrire la factorisation complète

    $P(x) = a(x - r)(x - x_1)(x - x_2)$ où $a$ est le coefficient dominant et $x_1$, $x_2$ les racines de $Q$.

  5. Vérifier

    Développer ou substituer une valeur test pour s’assurer que l’égalité est correcte.

Exemple complet

Factoriser $P(x) = 2x^3 - 5x^2 - 14x + 15$ sachant que $x = 3$ est racine.

  1. Division : $P(x) = (x - 3)(2x^2 + x - 5)$.
  2. Résolution du trinôme : $\Delta = 1 + 40 = 41$. \[ x = \frac{-1 \pm \sqrt{41}}{4}. \]
  3. Factorisation finale : $P(x) = 2(x - 3)\left(x - \frac{-1 + \sqrt{41}}{4}\right)\left(x - \frac{-1 - \sqrt{41}}{4}\right)$.

Autre exemple

$P(x) = x^3 - 4x^2 - x + 4$ avec racine évidente $x = 4$ :

  • Division : $P(x) = (x - 4)(x^2 - x - 1)$.
  • Résolution : $x = \dfrac{1 \pm \sqrt{5}}{2}$.
  • Factorisation : $(x - 4)\left(x - \dfrac{1 + \sqrt{5}}{2}\right)\left(x - \dfrac{1 - \sqrt{5}}{2}\right)$.

Conseils pratiques

  • Tester les racines évidentes parmi les diviseurs de la constante.
  • Utiliser Horner pour réduire les erreurs de calcul.
  • Après factorisation, évaluer $P(0)$ ou $P(1)$ pour vérifier la cohérence rapide.
  • Si le trinôme quotient n’a pas de racines réelles, mentionnez que la factorisation réelle s’arrête à $(x - r)Q(x)$.

Erreurs fréquentes

  1. Diviser dans le mauvais sens

    La division doit être $P(x) : (x - r)$ et non l’inverse.

  2. Oublier le coefficient dominant

    Le trinôme quotient doit conserver le coefficient $a$ du terme en $x^3$.

  3. Arrêt prématuré

    Ne pas résoudre $Q(x) = 0$ prive de la factorisation complète.
