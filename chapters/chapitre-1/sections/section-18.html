Chapitre 1 · Section 18

Équation paramétrique du second degré

Étudier une équation du second degré dépendant d’un paramètre, réaliser une disjonction de cas et décrire le nombre de solutions en fonction de la valeur du paramètre.

Accueil › Chapitre 1 › Section 18

Objectifs de la section

  • Exprimer le discriminant en fonction du paramètre.
  • Discuter le signe du discriminant pour déterminer le nombre de solutions.
  • Donner les solutions en fonction du paramètre lorsqu’elles existent.
  • Interpréter graphiquement l’évolution en fonction du paramètre.

Démarche standard

  1. Mettre l’équation sous forme standard

    Regrouper les termes pour obtenir $a(p)x^2 + b(p)x + c(p) = 0$ où $p$ est le paramètre.

  2. Calculer le discriminant

    $\Delta(p) = b(p)^2 - 4a(p)c(p)$.

  3. Étudier le signe de $\Delta(p)$

    Résoudre $\Delta(p) > 0$, $\Delta(p) = 0$, $\Delta(p) < 0$ pour déterminer les intervalles de $p$.

  4. Donner les solutions

    Pour $\Delta(p) \geq 0$, exprimer $x_{1,2}(p) = \dfrac{-b(p) \pm \sqrt{\Delta(p)}}{2a(p)}$.

  5. Analyser les cas limites

    Vérifier que $a(p) \neq 0$ et traiter séparément les valeurs du paramètre qui rendent $a(p) = 0$.

Exemple guidé

Résoudre $(p - 1)x^2 + 2px + (p + 3) = 0$ selon $p$.

  1. Cas $p = 1$ : l’équation devient $2x + 4 = 0$ d’où $x = -2$.
  2. Cas $p \neq 1$ : $a(p) = p - 1$, $b(p) = 2p$, $c(p) = p + 3$.
  3. $\Delta(p) = 4p^2 - 4(p - 1)(p + 3) = 4(p^2 - p^2 - 2p + 3) = 4(-2p + 3)$.
  4. Disjonction :
    • $\Delta(p) > 0 \Leftrightarrow p < \dfrac{3}{2}$ : deux solutions.
    • $\Delta(p) = 0 \Leftrightarrow p = \dfrac{3}{2}$ : une solution double.
    • $\Delta(p) < 0 \Leftrightarrow p > \dfrac{3}{2}$ : aucune solution réelle.
  5. Pour $p < \dfrac{3}{2}$, \[ x_{1,2} = \frac{-2p \pm 2\sqrt{-2p + 3}}{2(p - 1)} = \frac{-p \pm \sqrt{-2p + 3}}{p - 1}. \]

Interprétation graphique

Représenter la parabole $y = (p - 1)x^2 + 2px + (p + 3)$ pour différentes valeurs de $p$ montre comment le sommet et les intersections évoluent.

Conseils pratiques

  • Isoler les valeurs de $p$ qui annulent $a(p)$ avant de calculer $\Delta(p)$.
  • Factoriser $\Delta(p)$ pour lire plus facilement le signe.
  • Présenter la disjonction sous forme de tableau récapitulatif dans les rédactions longues.
  • Vérifier les solutions obtenues en les substituant dans l’équation initiale.

Erreurs fréquentes

  1. Oublier un cas particulier

    Si $a(p) = 0$, l’équation devient du premier degré : traitez-la à part.

  2. Confondre paramètre et inconnue

    Le paramètre $p$ est fixe dans chaque cas ; seules les inconnues $x$ sont résolues.

  3. Résultats non simplifiés

    Simplifiez les expressions de $x_{1,2}(p)$ pour éviter les erreurs.
