Chapitre 1 · Section 10

Existence de racines (forme canonique)

Discuter le nombre de solutions de $f(x) = 0$ à partir de $f(x) = a(x - \alpha)^2 + \beta$ sans recalculer le discriminant.

Accueil › Chapitre 1 › Section 10

Méthodes et exemples

Trouver les racines revient à résoudre l’équation : $$f(x) = a(x - \alpha)^2 + \beta = 0$$

On isole le carré : $$a(x - \alpha)^2 + \beta = 0 \quad \Rightarrow \quad (x - \alpha)^2 = -\frac{\beta}{a}$$

Le nombre de solutions dépend du signe de $-\frac{\beta}{a}$ :

  1. Si $-\frac{\beta}{a} < 0$ : aucune solution réelle → pas de racines.
  2. Si $-\frac{\beta}{a} = 0$ : une seule solution → une racine (double).
  3. Si $-\frac{\beta}{a} > 0$ : deux solutions → deux racines réelles.

  1. Exemple où $-\frac{\beta}{a} = 0$ :

    $$f(x) = 3(x - 4)^2 + 0$$

    On identifie $a = 3$, $\alpha = 4$, $\beta = 0$

    Résolution : \begin{align*} 3(x - 4)^2 = 0 \\ (x - 4)^2 = 0 \\ x - 4 = 0 \\ x = 4 \end{align*}

    Il y a une racine double : $x = 4$

  2. Exemple où $-\frac{\beta}{a} > 0$ :

    $$f(x) = -(x - 2)^2 + 9$$

    On identifie $a = -1$, $\alpha = 2$, $\beta = 9$

    Résolution : \begin{align*} -(x - 2)^2 + 9 = 0 \\ (x - 2)^2 = 9 \\ x - 2 = \pm \sqrt{9} \\ x = 2 \pm 3 \end{align*}

    Les racines sont donc : $$x_1 = 5, \quad x_2 = -1$$
