Chapitre 1 · Section 14

Résolution d’équations du second degré

Résoudre une équation du second degré $ax^2 + bx + c = 0$ sans changement de variable.

Accueil › Chapitre 1 › Section 14

Méthodes et exemples

On peut résoudre une équation du second degré de deux façons complémentaires :

  • graphiquement, en repérant les points où la courbe coupe l’axe des abscisses ;
  • par le calcul, à l’aide du discriminant $\Delta$ ou de certaines astuces selon les cas.

Résolution par le calcul

On peut résoudre $f(x) = 0$ en appliquant différentes méthodes selon la forme de l’expression :

  1. Cas régulier (discriminant)

    Soit : $f(x) = -x^2 + 4x + 5$

    On calcule le discriminant :

    \[ \Delta = 4^2 - 4(-1)(5) = 16 + 20 = 36 \]

    Les racines sont données par :

    \[ x_1 = \frac{-4 + \sqrt{36}}{2(-1)} = -1,\quad x_2 = \frac{-4 - \sqrt{36}}{2(-1)} = 5 \]

  2. Cas particulier : $\Delta = 0$

    Soit : $f(x) = x^2 - 2x + 1$

    On calcule le discriminant :

    \[ \Delta = (-2)^2 - 4(1)(1) = 4 - 4 = 0 \]

    La racine double est :

    \[ x = \frac{-(-2)}{2(1)} = 1 \]

  3. Cas particulier : $a + b + c = 0$

    Soit : $f(x) = 2x^2 - 3x + 1$

    On remarque que :

    \[ a + b + c = 2 - 3 + 1 = 0 \]

    On peut donc directement conclure :

    \[ x_1 = 1,\quad x_2 = \frac{c}{a} = \frac{1}{2} \]

  4. Cas particulier : $a - b + c = 0$

    Soit : $f(x) = 2x^2 + 3x + 1$

    On vérifie :

    \[ a - b + c = 2 - 3 + 1 = 0 \]

    On peut donc écrire :

    \[ x_1 = -1,\quad x_2 = -\frac{c}{a} = -\frac{1}{2} \]
